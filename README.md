## 📁 프로젝트 구조

```
src/
├── components/          # 재사용 가능한 컴포넌트
│   ├── layout/         # 레이아웃 컴포넌트

├── stores/             # Zustand 상태 관리
└── styles/
```

### 설치

```bash

corepack enable && corepack prepare pnpm --activate
pnpm i
pnpm dev
```
